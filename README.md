# CountStack

An application to keep track of multiple things simultaneously

# Features
- Save counters to localStorage for persistence
- Assign colors to each counter
- Provides undo and redo functionality
- Export the data in JSON format

# Folder Structure
```bash
└───src
    │─── App.jsx  # Source Code
    ├───components
    │   ─ CountersButton.jsx # Counter Button Component
    │   ─ ExportButton.jsx   # Export Button Component


```
